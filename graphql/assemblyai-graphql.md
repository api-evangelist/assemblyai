# AssemblyAI GraphQL API

AssemblyAI is an AI speech-to-text and audio intelligence API. The API covers async and real-time transcription, speaker diarization, sentiment analysis, topic detection, PII redaction, chapter detection, and LeMUR (LLM over audio).

**Endpoint:** No native GraphQL endpoint — conceptual schema derived from AssemblyAI REST/gRPC API
**Documentation:** https://www.assemblyai.com/docs/

- Reference: https://www.assemblyai.com/docs/api-reference/overview
- Schema: https://github.com/api-evangelist/assemblyai
