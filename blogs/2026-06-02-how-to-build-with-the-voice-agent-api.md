The Voice Agent API is a single WebSocket that handles the complete voice agent pipeline — speech-to-text, LLM reasoning, text-to-speech, turn detection, tool calling, and session resumption — at $4.50 per hour. Developers send audio input and receive audio output, eliminating the need to integrate separate providers for each component of the conversation system.
